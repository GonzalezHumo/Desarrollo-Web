1

Modificar contenido HTML

element.innerHTML ★ Método más usado
// Cambia el HTML interno de un elemento — permite etiquetas
document.getElementById("perfil").innerHTML =
  "<b> Nombre:</b> Victoria Marbella González Humo<br>" +
  "<b> Edad:</b> 20 años<br>" +
  "<b> Email:</b> marbellahumo@gmail.com<br>" +
  "<b> Ciudad:</b> Los Mochis, Sinaloa";
innerHTMLInserta etiquetas HTML — permite texto con formato enriquecido
2

Modificar texto plano

element.innerText Solo texto, sin HTML
// Cambia el texto del elemento — NO interpreta etiquetas HTML
document.getElementById("datos-texto").innerText =
  "Nombre : Victoria Marbella González Humo\n" +
  "Edad : 20 años\n" +
  "Email : marbellahumo@gmail.com\n" +
  "Ciudad : Los Mochis, Sinaloa";
innerTextSolo texto plano — las etiquetas HTML no se procesan
3

Escribir en el documento

document.write() ⚠ Solo para pruebas
// Escribe en el flujo del documento durante la carga
// ⚠ Usado DESPUÉS de cargar: borra TODO el HTML de la página

document.write("<p>Nombre: Victoria Marbella González Humo</p>");
document.write("<p>Edad : 20 años</p>");
document.write("<p>Email : marbellahumo@gmail.com</p>");
document.write("<p>Ciudad: Los Mochis, Sinaloa</p>");
document.write()Método clásico — si se ejecuta tarde, sobrescribe toda la página
⚠ Advertencia: usar document.write() después de que la página cargó elimina todo el HTML existente.

El contenido que habría escrito es:

<p>Nombre: Victoria Marbella González Humo</p>

<p>Ciudad: Los Mochis, Sinaloa</p>

Cuadro de diálogo emergente

  "Nombre : Victoria Marbella González Humo\n" +
  "Edad : 20 años\n" +
  "Email : marbellahumo@gmail.com\n" +
  "Ciudad : Los Mochis, Sinaloa"
);
window.alert()Abre un cuadro emergente nativo — bloquea la página hasta cerrarlo
